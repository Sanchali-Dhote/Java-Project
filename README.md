# Online Course Registration System

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Collections Framework (HashMap, ArrayList)

## Project Description
Developed a console-based Online Course Registration System using Java and OOP concepts.  
The system allows students to register for courses, view course details, manage enrollments, and generate fee invoices through a menu-driven interface.

Implemented features such as student registration, course enrollment, viewing registered courses, removing students, and invoice generation.  
Used Java Collections like HashMap and ArrayList for efficient data storage and management without using a database.

Applied OOP concepts including classes, objects, encapsulation, constructors, composition, method overriding, and exception handling to build a structured and maintainable application.

## Features
- Add and manage student records
- Display available courses
- Register students for multiple courses
- View student course details
- Remove student records
- Generate fee invoice
- Menu-driven console interface
